# Python-For-CyberSecurity
# LAB 3 ‚Äì Python untuk Cyber Security

## Deskripsi

Lab ini bertujuan untuk mempelajari penggunaan Python dalam bidang keamanan siber.
Beberapa script dibuat untuk melakukan otomatisasi tugas, proses kriptografi sederhana, pemindaian kerentanan dasar, serta pengambilan data dari website.

Melalui lab ini diharapkan mahasiswa memahami bagaimana Python dapat digunakan sebagai alat bantu dalam aktivitas keamanan siber dan pengumpulan informasi.

---

## Tujuan

Tujuan dari praktikum ini adalah:

1. Memahami penggunaan Python dalam bidang cyber security
2. Membuat script otomatisasi sederhana
3. Mengimplementasikan fungsi kriptografi dasar
4. Membuat scanner sederhana untuk mendeteksi potensi kerentanan
5. Mengambil informasi dari website menggunakan teknik web scraping

---

## Penjelasan Script

### automate.py

Script ini digunakan untuk melakukan otomatisasi beberapa tugas sederhana yang berkaitan dengan aktivitas keamanan siber.
Automasi ini bertujuan untuk mempermudah pekerjaan yang dilakukan secara berulang.

### scanner.py

Script ini berfungsi sebagai vulnerability scanner sederhana.
Program akan melakukan pemeriksaan dasar terhadap target untuk mendeteksi potensi kerentanan.

### scraper.py

Script ini digunakan untuk melakukan web scraping.
Program akan mengambil informasi dari halaman website yang kemudian dapat dianalisis lebih lanjut.

### crypto_tool.py

Script ini berisi fungsi kriptografi sederhana seperti proses enkripsi dan dekripsi.
Digunakan untuk memahami konsep dasar keamanan data.

### common.py

File ini berisi fungsi atau komponen umum yang dapat digunakan oleh script lain dalam proyek ini.

## Cara Menjalankan Program

1. Clone repository dari GitHub

```
git clone https://github.com/Fayedyamani107/Python-For-CyberSecurity.git
```

2. Masuk ke folder LAB_3

```
cd Python-For-CyberSecurity/LAB_3
```

3. Jalankan script yang diinginkan

Contoh:

```
python src/scanner.py
```

atau

```
python src/scraper.py
```
